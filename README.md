## 🚀 Features

- **Modern Dark Portfolio** — Elegant, professional single-page experience
- **Full PWA Support** — Installable, offline-ready with service worker & auto-update
- **Bilingual SEO** — Optimized for both English and Bangla search
- **Advanced Structured Data** — Person + WebSite schema with hreflang support
- **WebP Image Optimization** — Modern `<picture>` element with PNG fallback
- **Responsive Excellence** — LinkedIn-style wide layout on desktop, compact on mobile
- **Production Ready** — Clean build pipeline with Nitro + TanStack Start

---

## 🛠 Tech Stack

- **Framework**: TanStack Start (React + SSR)
- **Styling**: Tailwind CSS v4 + shadcn/ui
- **PWA**: vite-plugin-pwa + Workbox
- **Build Tool**: Vite + Nitro
- **Language**: TypeScript
- **SEO**: Full JSON-LD, Open Graph, Twitter Cards, sitemap, hreflang

---

## 💻 Local Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Run tests
pnpm test

# Build for production
pnpm build

# Preview production build
npx vite preview
```

---

## 📦 Deployment

This project is built with **TanStack Start + Nitro** and can be easily deployed to:

- Vercel
- Netlify
- Cloudflare Pages
- Any Node.js compatible host

The build outputs to `.output/` directory.
